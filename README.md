# LaserBotGui
GUI for move and controll laser-bot

Technologies:
1. C++ language
2. Qt + opencascade
3. qmake build system

Build:
1. Add CSF_OCCTIncludePath - path to opencascade include files;
   example: /usr/include/opencascade
2. Add CSF_OCCTLibPath - path to opencascade lib files;
   example: /usr/lib
